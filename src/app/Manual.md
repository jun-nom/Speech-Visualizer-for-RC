# スピーチフロー可視化ツール - 使用マニュアル

## 概要

スピーチフロー可視化ツールは、カンファレンスのオンライン配信やYoutube Liveなどの音声書き起こしテキストを構造化し、Miro上での可視化に適した形式に変換するWebアプリケーションです。リアルタイム音声書き起こしツールで得られたテキストを2〜5分間隔で入力し、OpenAI APIを活用してトピック単位に整理・要約します。

## 基本的なワークフロー

1. **音声書き起こし**: Zoomの書き起こし機能やその他のリアルタイム音声書き起こしツールでカンファレンス・ライブ配信の内容をテキスト化
2. **テキスト入力**: 2〜3分（最大5分）間隔で書き起こしテキストを本アプリにコピーペースト
3. **自動構造化**: OpenAI APIがテキストをトピック単位に分析・要約し、3種類のノードで可視化
4. **Miroへ転記**: 生成された構造化コンテンツをMiroにコピーして最終的なレイアウト調整
5. **レイアウト調整**: Miro上でスピーチの流れがわかりやすくなるよう配置を調整

## 主な機能

### 1. リアルタイム音声書き起こし対応
- **段階的テキスト入力**: 2〜3分間隔（最大5分）で書き起こしテキストを順次入力
- **トピック単位の区切り**: 発表者の話題転換に合わせたテキスト分割を推奨
- **コピー&ペースト最適化**: 音声書き起こしツールからの効率的なテキスト転記

### 2. 自動構造化・可視化
- **OpenAI API解析**: 高精度なテキスト解析によるトピック抽出
- **3種類のノード自動生成**:
  - **タイトルノード**: 薄い水色背景 + 青枠 + 太字（トピックの主題）
  - **事実ノード**: 白背景 + 青い点線枠（客観的情報・データ）
  - **洞察ノード**: #3640A5背景 + 白文字（主観的考察・分析）
- **80文字制限**: 1ノードあたり80文字以内、超過時は自動分割

### 3. Miro連携最適化
- **コピー対応フォーマット**: 生成されたノード内容をMiroに転記しやすい形式で表示
- **視覚的レイアウト**: Miroでの配置を想定した論理的なノード配列
- **段階的更新**: リアルタイムでコンテンツが追加され、随時Miroに反映可能

### 4. セッション管理
- **カンファレンス単位管理**: 各配信・カンファレンスごとに独立したセッション作成
- **履歴保存**: 過去のカンファレンス内容の振り返りと再利用
- **リアルタイム同期**: Supabaseによる自動保存とデータ永続化

### 5. 感想・質問生成
- **コンテンツ理解促進**: 配信内容から感想と質問を自動生成
- **議論活性化**: カンファレンス後の討議や質疑応答のサポート
- **フォローアップ支援**: 追加で検討すべき観点の提示

### 6. オフライン対応
- **接続不安定対応**: カンファレンス会場などでの接続問題に対応
- **ローカルバックアップ**: データの自動ローカル保存
- **接続復旧時同期**: オンライン復帰時の自動データ同期

## ノード構成の詳細

### 基本構造
各トピックは原則として3つのノードで構成されます：

1. **タイトルノード（1個）**
   - トピックの主題を表示
   - 薄い水色背景 + 青枠 + 太字スタイル

2. **事実ノード（1個以上）**
   - 客観的情報・データ・事実を表示
   - 白背景 + 青い点線枠スタイル

3. **洞察ノード（1個以上）**
   - 主観的考察・分析・解釈を表示
   - #3640A5背景 + 白文字スタイル

### 文字数制限
- **基本制限**: 1ノードあたり80文字以内
- **分割ルール**: 80文字を超える場合のみ複数ノードに分割
- **最適化**: 読みやすさを重視した自動分割

## 使用上の注意

### ノートPC環境での最適利用
- **入力頻度**: 2〜3分に一回のペースで書き起こしテキストを入力（最大5分間隔）
- **トピック区切り**: 発表者の話題転換やセクション変更に合わせたテキスト分割
- **タブ切り替え効率化**: ブラウザタブでの配信視聴、書き起こし、本アプリ、Miroの切り替え
- **ウィンドウ分割活用**: 可能であれば配信視聴と本アプリを左右分割表示

### データの取り扱い
- **配信内容の機密性**: カンファレンス内容の適切な管理とアクセス権限の確認
- **著作権の配慮**: 配信内容の二次利用における権利関係の確認
- **APIキー管理**: OpenAI APIキーの安全な保管

### ノートPC環境でのMiro連携
- **効率的なタブ操作**: キーボードショートカット（Ctrl+Tab等）を活用した高速タブ切り替え
- **段階的転記**: ノード生成後、Miroタブに切り替えて転記、再び本アプリに戻る
- **ブックマーク活用**: 頻繁にアクセスするMiroボードをブックマークして素早くアクセス
- **ウィンドウ最小化/最大化**: 作業に集中したいツールを最大化、他は最小化
